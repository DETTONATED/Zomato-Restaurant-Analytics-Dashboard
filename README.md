# Zomato Restaurant Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes Zomato restaurant data to uncover insights about ratings, cuisines, online delivery services, table booking availability, and customer engagement.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Dashboard Preview

![Dashboard](dashboard.png)

## Key Insights
- North Indian cuisine is the most popular cuisine.
- 74% of restaurants provide online delivery.
- Most restaurants do not offer table booking.
- Average restaurant rating is 2.67.
